# Gravy59 Fonts

## How do I install these formulae?

`brew install gravy59/fonts/<formula>`

Or `brew tap gravy59/fonts` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "gravy59/fonts"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
